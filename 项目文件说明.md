## 📄 文件详细说明

### 🔴 核心文件（必需）

#### `index.html` 
**用途**: 应用的主界面文件  
**重要性**: ⭐⭐⭐⭐⭐（最重要）  
**说明**: 
- 包含所有页面结构
- 定义了7个不同的界面（主界面、呼吸、情绪选择、3个干预模式、完成）
- 双击即可在浏览器中打开使用
- 约170行HTML代码

**使用方法**:
```
直接双击打开 或 右键→打开方式→选择浏览器
```

---

#### `style.css`
**用途**: 控制应用的外观和动画  
**重要性**: ⭐⭐⭐⭐⭐（必需）  
**说明**:
- 定义所有颜色、字体、布局
- 包含呼吸球、按钮等动画效果
- 响应式设计代码
- 约600行CSS代码

**主要样式**:
- 渐变背景色
- 按钮动画效果
- 呼吸球放大缩小
- 页面切换过渡

---

#### `app.js`
**用途**: 应用的交互逻辑  
**重要性**: ⭐⭐⭐⭐⭐（必需）  
**说明**:
- 控制页面跳转
- 管理呼吸动画
- 处理按钮点击
- 存储使用次数
- 约250行JavaScript代码

**核心功能**:
- 屏幕切换函数
- 呼吸练习逻辑
- 情绪选择处理
- 数据持久化

---

#### `icon.svg`
**用途**: 应用图标  
**重要性**: ⭐⭐（可选，但推荐）  
**说明**:
- 显示在浏览器标签页
- 添加到主屏幕时的图标
- 矢量图，支持任意缩放
- 心形+十字组合设计

## 🔧 常见任务指南

### 任务1: 第一次使用
```
步骤:
1. 阅读 "快速开始.txt"
2. 双击 "index.html" 或 "启动服务器.bat"
3. 在浏览器中体验
```

---

### 任务2: 在手机上安装
```
步骤:
1. 启动本地服务器（用启动服务器.bat）
2. 查看电脑IP地址（ipconfig命令）
3. 手机连接相同WiFi
4. 手机浏览器访问 http://[电脑IP]:8000
5. 添加到主屏幕
```

---

### 任务3: 修改颜色
```
步骤:
1. 用编辑器打开 "style.css"
2. 搜索需要修改的颜色（如 #667eea）
3. 替换为新颜色
4. 保存后刷新浏览器查看效果
```

---

### 任务4: 修改文字
```
步骤:
1. 用编辑器打开 "index.html"
2. 搜索要修改的文字
3. 直接修改
4. 保存后刷新浏览器
```

---

### 任务5: 测试功能
```
步骤:
1. 打开 "测试清单.md"
2. 在浏览器中打开应用
3. 按照清单逐项测试
4. 记录发现的问题
```

---

### 任务6: 准备扩展开发
```
步骤:
1. 通读 "需求文档.md"
2. 查看 "开发总结.md" 的后续计划
3. 阅读 "MVP-简化需求.md" 了解当前限制
4. 开始编码
```

## ⚠️ 重要提示

### 不要删除的文件
- ✋ `index.html` - 删了就不能用了
- ✋ `style.css` - 删了界面会乱
- ✋ `app.js` - 删了功能会失效

### 可以删除的文件（不影响运行）
- ✅ 所有 `.md` 文件（文档）
- ✅ `.txt` 文件（说明）
- ✅ `.bat` 文件（启动脚本）
- ✅ `icon.svg`（图标，但不建议删）

### 修改建议
- 💡 修改前先备份
- 💡 一次只改一个文件
- 💡 改完立即测试
- 💡 不确定就看文档
