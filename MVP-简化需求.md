# 情绪急救 App - MVP 最小功能集

## 🎯 MVP核心功能（只做这些）

### 1. 主界面
- 一个大红色按钮"紧急求助"

### 2. 呼吸引导（30秒）
- 简单的圆形动画（放大缩小）
- 文字提示"跟着呼吸"
- 30秒倒计时

### 3. 情绪分类
- 3个大按钮：恐慌、暴怒、悲伤

### 4. 干预页面（极简版）
- **恐慌**: 显示5-4-3-2-1引导文字
- **暴怒**: 可以拖动滑块或点击屏幕释放
- **悲伤**: 显示安慰文字

### 5. 完成页面
- 简单的"你做到了"反馈

## 技术实现
- 纯Web应用（HTML + CSS + JavaScript）
- 单个HTML文件即可运行
- 无需后端、无需安装
- 可以直接在手机浏览器打开

## 不包含的功能
❌ 可穿戴设备
❌ 语音播放
❌ 数据存储
❌ 复杂3D动画
❌ 用户账号系统
