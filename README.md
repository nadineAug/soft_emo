# 情绪急救 App (EmotionAid) - MVP版本

## 🎯 项目简介

这是一款极简的情绪危机干预应用，帮助用户在恐慌、暴怒或悲伤等情绪危机时刻快速稳定状态。

## ✨ 核心功能

1. **红色警报按钮** - 一键启动紧急模式
2. **30秒呼吸引导** - 强制生理缓冲
3. **情绪分类** - 恐慌、暴怒、悲伤三种选择
4. **定向干预**:
   - 恐慌 → 5-4-3-2-1 感官接地练习
   - 暴怒 → 滑块/点击宣泄
   - 悲伤 → 安慰文字显示

## 💡 使用说明

### 1. 紧急模式触发
- 点击首页的红色"紧急求助"按钮
- 系统立即进入呼吸引导

### 2. 呼吸练习
- 跟随屏幕上的呼吸球进行呼吸
- 吸气4秒 → 屏息2秒 → 呼气6秒
- 共3个周期，约30秒

### 3. 情绪分类
- 选择当前最强烈的情绪
- 系统会提供对应的干预方法

### 4. 干预练习
- **恐慌**: 完成5步感官接地练习
- **暴怒**: 滑动滑块至80%或点击30次
- **悲伤**: 阅读安慰文字，点击"我感觉好一点了"

### 5. 完成
- 查看使用统计
- 可拨打心理援助热线
- 返回首页
