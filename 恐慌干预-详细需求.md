# 恐慌干预功能 - 详细需求文档

## 🎯 功能概述
当用户点击紧急按钮后，启动完整的恐慌干预流程，通过分阶段引导帮助用户度过恐慌情绪。

---

## 📱 触发入口
- **触发方式**：用户点击主界面的"紧急求助"按钮
- **进入流程**：立即进入阶段1（生理锚定）

---

## 🔄 完整干预流程

### 阶段1：生理锚定（60秒）
**目标**：通过呼吸调节帮助用户稳定生理状态

**实现细节**：
- **视觉**：
  - 背景：统一视觉规范（深暖紫色背景 + 星光粒子 + 暖光渐变）
  - 显示呼吸球动画（圆形放大缩小）
  - 呼吸节奏：4-7-8（吸气4秒，屏息7秒，呼气8秒）
  - 视觉引导：呼吸球跟随节奏变化大小
  - 文字提示："跟着呼吸球，吸气...屏息...呼气..."（使用统一字体和颜色）
  - 倒计时显示：60秒倒计时
- **语音**：
  - 可选的呼吸引导语音（温和的女声）

**交互**：
- 右下角极小按钮："跳过这个，进入下一步"
- 长按屏幕：调出"我想休息"选项（直接跳到阶段7）

---

### 阶段2：安全确认（15秒）
**目标**：通过认知重构帮助用户确认当前安全状态

**实现细节**：
- **视觉**：
  - **文字显示**：
    - 屏幕中央大字显示："你此刻是安全的。这不是真正的危险。"
    - 使用统一的文字动态效果（逐句淡入 + 逐词淡入）
    - 逐句淡入：每句话间隔 0.3秒
    - 逐词淡入：每个词出现时带有轻微的放大效果（0.95→1.0）和模糊到清晰的过渡（blur 3px→0px）
    - 每句话完整显示后停留 2.5秒，然后下一句开始淡入，当前句保持显示（累积）
    - 阶段结束时，所有文字在 1秒内统一淡出
  - **字体样式**：
    - 字体：圆润的手写体（Kalam），柔和的米白色（#F5E6D3）
    - 每个文字带有微弱的、周期 2秒的呼吸光晕（text-shadow 从 0 到 4px 缓慢变化）
    - 光晕颜色：暖白色（rgba(255, 250, 240, 0.6)）
  - **背景**：统一视觉规范（深暖紫色背景 + 星光粒子 + 光晕流动）
    - 从阶段1的呼吸球淡出时，背景逐渐过渡为深暖紫色（#2D1B3A）
    - 星光粒子：5-8个，大小2-6px，透明度0.1-0.3，30秒循环
    - 光晕流动：暖橙色（#FF6B35），透明度0.05，8秒周期
- **语音**：
  - 自动播放语音："你此刻是安全的。这不是真正的危险。"
  - 如果用户有录制个人版本：优先播放"我的声音"版本
  - 如果没有录制：播放系统默认语音

**交互**：
- 右下角极小按钮："跳过这个，进入下一步"
- 长按屏幕：调出"我想休息"选项

## 🎨 通用交互规则

### 1. 跳过功能
- **位置**：每个阶段右下角
- **样式**：极小按钮，不干扰主要内容
- **功能**：跳过当前阶段，进入下一阶段

### 2. 长按休息
- **触发**：长按屏幕任意位置
- **功能**：调出"我想休息"选项
- **结果**：直接跳到阶段7（结束与见证）

### 3. 语音播放
- 所有语音提示应清晰、温和、有节奏
- 支持用户录制个人版本（阶段2）
- 如果用户未录制，使用系统默认语音

### 4. 视觉设计（统一规范）
**所有阶段统一使用以下视觉风格，营造温和、安全、有安全感的氛围：**

- **背景**：
  - 深暖紫色渐变：`#2D1B3A` 到 `#1a0f1f`（从上到下）
  - **星光粒子**（所有阶段统一）：
    - 数量：5-8个粒子
    - 大小：随机，范围 2-6px
    - 透明度：在 0.1-0.3 之间缓慢变化
    - 移动路径：随机曲线轨迹
    - 移动速度：极慢，每30秒完成一个来回循环
    - 视觉效果：极淡，几乎察觉不到，但增加层次感，营造"静谧夜空"的感觉
    - 颜色：暖色调，与背景协调（米白色或淡暖橙色）
  - **光晕流动**（所有阶段统一）：
    - 位置：屏幕边缘
    - 方向：从上到下缓慢流动
    - 周期：8秒完成一个循环
    - 颜色：暖橙色 `#FF6B35`
    - 透明度：仅 0.05（几乎不可察觉，仅增加层次感）
    - 效果：非常微弱，不干扰主要内容
  - 从阶段1过渡到阶段2时，背景平滑淡入（1.5秒过渡动画）

- **字体**：
  - 字体家族：圆润的手写体（优先使用"Kalam"字体，备选：Comic Sans MS、微软雅黑）
  - 字体颜色：柔和的米白色 `#F5E6D3`（不是刺眼的白）
  - 字体大小：根据内容重要性，主要文字 36-48px，次要文字 24-28px
  - **文字动态效果**（所有阶段统一，重要文字使用）：
    - **出现方式**：
      - 逐句淡入：每句话之间间隔 0.3秒
      - 逐词淡入：每句话内的文字采用逐词淡入（如："我知道…你…现在…很害怕"）
      - 每个词出现时带有轻微的放大效果：从 scale(0.95) 到 scale(1.0)
      - 模糊到清晰的过渡：从 filter: blur(3px) 到 filter: blur(0px)
      - 营造"正在被诉说"的亲密感
    - **停留时长**：
      - 每句话完整显示后停留 2.5秒
      - 然后下一句开始淡入
      - 当前句保持显示（累积显示，不消失）
    - **消失方式**：
      - 阶段结束时，所有文字在 1秒内统一淡出
      - 同时下一阶段的引导语开始淡入
    - **字体动画**：
      - 每个文字带有微弱的、周期 2秒的呼吸光晕
      - text-shadow 从 0 到 4px 缓慢变化
      - 光晕颜色：暖白色（rgba(255, 250, 240, 0.6)）
      - 使用 CSS animation 实现平滑的呼吸效果

- **动画**：
  - 所有过渡动画：流畅、舒缓、不突兀（0.8秒过渡）
  - 背景淡入：1.5秒平滑过渡
  - **文字动画**：
    - 逐句淡入：每句话间隔 0.3秒
    - 逐词淡入：每个词间隔 300ms（可配置）
    - 词出现动画：scale(0.95→1.0) + blur(3px→0px)，持续时间 0.5秒
    - 句停留时长：2.5秒
    - 阶段结束淡出：1秒统一淡出
  - **光晕呼吸动画**：
    - 周期：2秒
    - text-shadow 变化：0px → 4px（缓慢变化）
    - 光晕颜色：暖白色（rgba(255, 250, 240, 0.6)）
    - 与呼吸节奏同步

- **布局**：
  - 简洁、聚焦、减少干扰
  - 主要内容居中显示
  - 按钮和交互元素不干扰主要内容

## 📋 技术实现要点

### 前端实现
- **统一视觉系统**：
  - **背景组件**：
    - 深暖紫色渐变：CSS linear-gradient（#2D1B3A 到 #1a0f1f）
    - **星光粒子Canvas**：
      - 使用Canvas绘制，所有阶段共用同一实例
      - 粒子数量：5-8个（随机生成）
      - 粒子大小：2-6px（每个粒子随机）
      - 透明度：0.1-0.3（使用Math.sin实现缓慢变化）
      - 移动路径：使用贝塞尔曲线或正弦波实现随机曲线轨迹
      - 移动速度：极慢，30秒完成一个来回循环（使用requestAnimationFrame，每帧移动极小距离）
      - 颜色：暖色调（米白色或淡暖橙色），与背景协调
      - 实现方式：每个粒子独立计算位置和透明度，使用缓动函数实现平滑移动
    - **光晕流动层**：
      - 使用CSS linear-gradient实现
      - 颜色：暖橙色 #FF6B35
      - 透明度：0.05（rgba(255, 107, 53, 0.05)）
      - 动画：CSS animation，8秒周期，从上到下流动
      - 位置：屏幕边缘，使用绝对定位
  - **统一字体系统**：引入Kalam字体，所有文字使用米白色（#F5E6D3）
  - **文字动态效果实现**：
    - **逐句淡入**：
      - 使用JavaScript控制，每句话间隔0.3秒
      - 每句话作为独立元素，使用CSS transition实现淡入
    - **逐词淡入**：
      - 将每句话拆分为词（span元素）
      - 每个词延迟显示：index * 300ms
      - 使用CSS animation实现词出现动画：
        - scale: 0.95 → 1.0（0.5秒）
        - filter: blur(3px) → blur(0px)（0.5秒）
        - opacity: 0 → 1（0.5秒）
    - **句停留控制**：
      - 每句话完整显示后，使用setTimeout等待2.5秒
      - 下一句开始淡入时，当前句保持显示（不消失）
    - **阶段结束淡出**：
      - 所有文字元素统一添加fade-out类
      - CSS transition: opacity 1 → 0，duration 1秒
    - **呼吸光晕动画**：
      - 使用CSS @keyframes实现
      - text-shadow: 0px → 4px（2秒周期）
      - 光晕颜色：rgba(255, 250, 240, 0.6)
      - 应用到所有文字元素
  - **统一光晕效果**：使用CSS text-shadow实现多层光晕，配合2秒脉动动画
  - **统一过渡动画**：所有阶段切换使用0.8秒平滑过渡
- **语音播放**：
  - 使用Web Speech API实现语音播放
  - 自动选择温和的女声（优先微软小晓、小艺等）
  - 统一语音参数：rate=0.75, pitch=0.85（温和、有安全感）
- **动画实现**：
  - 使用Canvas实现星光粒子动画（所有阶段共用）
  - 使用CSS动画实现呼吸球、潮汐动画
  - **文字动画实现**：
    - 使用JavaScript控制逐句和逐词淡入时序
    - 使用CSS animation实现词出现动画（scale + blur + opacity）
    - 使用CSS @keyframes实现呼吸光晕动画（text-shadow变化）
    - 使用CSS transition实现阶段结束淡出效果
- **交互实现**：
  - 使用触摸事件实现思维擦拭交互
  - 使用LocalStorage存储用户预设（安全照片、个人语音）

### 计时与统计
- 记录每个阶段的开始和结束时间
- 计算总耗时并在阶段7显示
- 支持阶段跳过，不影响总计时

### 响应式设计
- 适配手机屏幕（竖屏为主）
- 按钮大小适合手指点击
- 文字大小适合阅读
